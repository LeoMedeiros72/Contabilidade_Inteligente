# Contabilidade_Inteligente
Um código que implementa machine learning e IA para criar um chatbox, como chatGPT, que responde perguntas relacionadas ao banco de dados que foi conectado. Isso demonstra o poder da IA + ML para reduzir tempo de procura por respostas simples e assim o foco pode ser em partes mais importantes. Na versão 1.0 é esperado que reduza até 15% do tempo
